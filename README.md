## 简介

ClipShelf 是一款为 macOS 设计的精美剪贴板管理工具，帮助您高效管理临时内容。它自动捕获您复制的所有内容，并以优雅的方式组织在浮层面板中，让粘贴变得前所未有的简单快捷。

## 核心功能

### 📝 自动捕获
- 自动捕获剪贴板内容（文本、图片、文件）
- 实时后台监控，无需手动操作
- 智能识别内容类型

### ⚡ 快速粘贴
- **双击**卡片即刻粘贴
- **⌘1-9** 快捷键直接粘贴前 9 项
- 无需切换窗口，效率倍增

### 🔍 智能组织
- 强大的搜索功能
- 内容类型筛选（文本/图片/文件）
- 彩色标签管理
- 置顶重要项目

### 📝 智能分词
- 中文智能分词功能
- 一键拆分文本为词块
- 精准选择需要的词汇

### 🔄 拖放支持
- 从 Finder 拖入文件
- 拖出到其他应用
- 无缝工作流

### 🔒 隐私优先
- **纯本地存储**，不上传任何数据
- 零外部依赖
- 无需注册账号

## 系统要求

- macOS 14.0 或更高版本
- Apple Silicon 或 Intel 处理器

## 权限说明

ClipShelf 需要以下系统权限：
