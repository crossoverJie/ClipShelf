1. 服务说明

ClipShelf 是一款 macOS 剪贴板管理工具，提供内容捕获、组织、快速粘贴等功能。基础功能免费使用，高级功能可通过订阅 ClipShelf Pro 解锁。

1. Service Description

ClipShelf is a macOS clipboard management tool that provides content capture, organization, and quick paste features. Basic features are free, while advanced features can be unlocked by subscribing to ClipShelf Pro.

2. 订阅服务

ClipShelf Pro 提供以下订阅选项：

  • 月度订阅：自动续费，可随时取消
  • 年度订阅：自动续费，可随时取消
  • 终身买断：一次性购买，永久使用

所有订阅均包含 7 天免费试用期。试用期结束后将自动开始计费，除非您在试用期结束前取消。

2. Subscription Services

ClipShelf Pro offers the following subscription options:

  • Monthly Subscription: Auto-renewing, can be cancelled anytime
  • Annual Subscription: Auto-renewing, can be cancelled anytime
  • Lifetime Purchase: One-time purchase, permanent access

All subscriptions include a 7-day free trial. Billing will automatically begin after the trial ends unless cancelled before.

3. 取消与退款

订阅管理通过 Apple ID 进行：

  • 可随时在系统设置 → Apple ID → 订阅中取消
  • 取消后当前订阅期结束前仍可继续使用 Pro 功能
  • 退款请求需通过 Apple 处理

3. Cancellation & Refunds

Subscription management is handled through your Apple ID:

  • Can be cancelled anytime in System Settings → Apple ID → Subscriptions
  • Pro features remain available until the end of the current billing period
  • Refund requests must be processed through Apple
